Zack Command¶

The zack command is used to interact with mesos and marathon clusters defined in your inventories.

zack¶

Usage: zack [OPTIONS] COMMAND [ARGS]...

  Manage your application in Marathon

Options:
  -v, --verbose    Make commands verbose, use multiple time for higher
                   verbosity
  --log-file TEXT  When using the verbose flag, redirects output to this file
  -h, --help       Show this message and exit.

Commands:
  destroy      Destroy your app.
  push         Push your app in Marathon.
  scale        Scale your app.
  show-config  Show the configuration of your app.
  show-tasks   Show the tasks running of your app.
  suspend      Suspend your app.
  tail         Display the logs of your app.

zack destroy¶

Usage: zack destroy [OPTIONS]

  Destroy your app.

Options:
  -i, --inventory TEXT  Inventory file to use to find the Marathon cluster.
  -a, --app TEXT        The application you want to manage.
  -g, --group TEXT      Group to use to find the Marathon cluster in the
                        inventory file.
  -h, --help            Show this message and exit.

zack push¶

Usage: zack push [OPTIONS]

  Push your app in Marathon.

Options:
  -C, --command TEXT       The command to run.
  -c, --cpus FLOAT         The number of CPUs to give to this app, can be a
                           fraction.
  -m, --mem INTEGER        The memory limit for this app, in MB, can be a
                           fraction.
  -n, --instances INTEGER  The number of instances to run.
  -p, --ports INTEGER      The ports you want to use for your application.
  -e, --env TEXT           Environment variable to define in Marathon.
  -i, --inventory TEXT     Inventory file to use to find the Marathon cluster.
  -a, --app TEXT           The application you want to manage.
  -g, --group TEXT         Group to use to find the Marathon cluster in the
                           inventory file.
  -h, --help               Show this message and exit.
